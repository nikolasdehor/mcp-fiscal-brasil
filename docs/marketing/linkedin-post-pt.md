Desenvolver software no Brasil exige lidar com uma das infraestruturas fiscais mais complexas do mundo. Para empresas que estão integrando inteligência artificial em seus processos, o acesso a dados de CNPJ, NFe e SPED sempre foi um gargalo técnico, exigindo integrações customizadas e manutenção constante.

É com satisfação que apresento o MCP Fiscal Brasil, o primeiro servidor Model Context Protocol (MCP) focado no sistema fiscal brasileiro.

O objetivo do projeto é simplificar a ponte entre assistentes de IA (como Claude, Cursor e IDEs) e os dados governamentais. Ao implementar o padrão MCP, permitimos que modelos de linguagem executem consultas e validações fiscais de forma nativa e estruturada.

O que o MCP Fiscal Brasil oferece hoje:
- Integração nativa com assistentes de IA para consulta de dados de CNPJ e Simples Nacional.
- Validação offline de chaves de acesso NFe e documentos como CPF/CNPJ.
- Monitoramento de status dos serviços SEFAZ estaduais em tempo real.
- Parsing e análise estrutural de arquivos SPED (EFD/ECD) para auditoria automatizada.

Além de funcionar como um servidor MCP, o projeto disponibiliza um SDK Python robusto, facilitando o uso dessas ferramentas em aplicações tradicionais como FastAPI, Django e Flask.

O projeto é de código aberto (MIT) e está em constante evolução. Nossa visão é expandir para emissão de notas fiscais e suporte a centenas de municípios para NFSe, tornando-se a camada de infraestrutura definitiva para IA fiscal no Brasil.
