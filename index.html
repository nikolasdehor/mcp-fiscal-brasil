Capacidades

Tudo do sistema fiscal brasileiro,
via linguagem natural

Integre consultas fiscais reais ao seu fluxo de trabalho com IA. Sem APIs complicadas, sem formatos XML manuais.

🏢
Consulta CNPJ
BrasilAPI como fonte primária com fallback automático para ReceitaWS. Dados completos da empresa em segundos.
🧾
NFe por Chave de Acesso
Consulta e parse de Notas Fiscais Eletrônicas direto do Portal da SEFAZ, com dados estruturados prontos para uso.
📡
Status SEFAZ em Tempo Real
Monitore a disponibilidade dos serviços da SEFAZ de qualquer estado. Essencial para sistemas de emissão em produção.
🟢
Simples Nacional
Verifique enquadramento e opção pelo Simples Nacional via CNPJ. Dados do portal oficial da Receita Federal.
📊
Análise SPED
Leitura e interpretação de arquivos SPED Fiscal e Contábil. Transforme blocos de texto em dados estruturados.
🏙️
Portal NFSe
Consulta de Notas Fiscais de Serviço Eletrônicas por município. Suporte ao padrão nacional NFSe.
👥
Eventos eSocial
Consulta e validação de eventos do eSocial. Suporte aos principais grupos de eventos de RH e folha.
🔐
Validação CPF/CNPJ Offline
Validação de dígitos verificadores de CPF e CNPJ sem chamadas de API. Rápida, gratuita e sem dependências externas.
🧩
SDK Mode
Use o pacote como biblioteca Python diretamente no seu código, sem precisar do servidor MCP. Ideal para scripts e integração ERP.

SDK Mode

Use como biblioteca Python

Sem servidor MCP. Importe diretamente no seu código para integração com ERPs, scripts de automação ou APIs internas.

python
from mcp_fiscal_brasil import FiscalBrasilSDK

sdk = FiscalBrasilSDK()

# Consultar empresa por CNPJ
empresa = await sdk.consultar_cnpj("11222333000181")
print(empresa.razao_social, empresa.situacao_cadastral)

# Validar CNPJ offline
valido = sdk.validar_cnpj("11.222.333/0001-81")

# Status SEFAZ por estado
status = await sdk.status_sefaz(uf="SP")
